Utility functions¶

This module provides various helper functions.

Functions¶

bkjd_to_astropy_time(bkjd[, bjdref]) Converts BKJD time values to an astropy.time.Time object.
btjd_to_astropy_time(btjd[, bjdref]) Converts BTJD time values to an astropy.time.Time object.
channel_to_module_output(channel) Returns a (module, output) pair given a CCD channel number.
module_output_to_channel(module, output) Returns the CCD channel number for a given module and output pair.
running_mean(data, window_size) Returns the moving average of an array data.
